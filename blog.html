Question-1: What are the different ways to select an element in the DOM?

By ID, By Class Name, By Tag Name

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets content inside an element, innerText sets visible text inside an element, textContent sets the text content of an element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where parent element handles events for multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is how events move in the DOM. When we click on an element, the event happens on that element first, then moves up to its parent, then its grandparent, and keeps going until it reaches the top of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

By adding an element to the dom, by removing an element to the dom, by replacing an element.
